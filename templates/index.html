ABOUT KWoC

Kharagpur Winter of Code is a 5-week long online program for the students , who are new to open source software development. The program not only helps students to get involved in open source, but also preps them for many open source summer programs, Google Summer of Code being one of them.

Why KWoC?

Intro to Open Source

KWoC provides a great opportunity to get acquainted with Github along with Git commands and contribute to open source effeciently.

Brush up your coding skills

If you love coding and want to learn about software development then KWoC helps you to get a glimpse of it and give you a head start.

Prepare for GSoC

With KWoC, you get to know how to select a project, interact with Mentors and learn all other things that prepares you in the best way for the next GSoC.

Timeline

KWoC Commences
November 2017

Kharagpur Winter of Code announced

KWoC Program is announced and we allow all mentor to register and submit projects
6thNov 2017

Accepting Student applications

Project and mentor registrations close while student registrations are open. Accepted projects are released.

15thNov 2017

Students paired up mentors

Students are paied with mentors and assigned projects. They can now start working towards KWoC.

1stDec 2017

Leaderboard Up

Leaderboard showing contributions of various students in KWoC is now visible to everyone.

7thDec 2017

Mid evaluations

Students and mentors have to provide mid evaluation reports to keep track of progress. Students with no commits would be removed from the program.

15thDec 2017

Last PRs

Last day to submit all of your KWoC related PRs

31stDec 2017

End evaluation report

Students have to provide a end evaluation report published in their blog to successfully pass the program.

7thJan 2018
KWOC ENDS
